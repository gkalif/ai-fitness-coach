\# AI Fitness Coach



Camera-based fitness app using pose detection to count reps and correct form in real time.



\## Features

\- Real-time pose detection via TensorFlow.js (MoveNet)

\- Hysteresis-based state machine for accurate rep counting

\- Lunge detection with reference animation overlay

\- Form feedback powered by Claude API



\## Tech Stack

TensorFlow.js · MoveNet · Claude API · React Native (in progress)



\## Status

Working HTML prototype functional. React Native build in progress.
